Feedback, Support & Bug Tracking

Does this require an official account?

Aurora supports both Microsoft official login for skins and servers, and Offline mode for local play.

Is login permanent?

Yes. In v1.1.0, session tokens are saved locally in accounts.json so you stay logged in permanently.

Why does it install Java 21?

Modern versions require it. Aurora provisions a portable JDK 21 automatically to keep your system clean.

Is the built-in X-Ray detectable?

Our X-Ray uses custom resource-pack injection. It is undetectable by standard client-side checks.
