Aurora Launcher is a personal project for playing Minecraft versions Beta 1.7.3 to 1.21.1. It features built-in tools like Fullbright, Modrinth browsing, custom themes, and was proudly built with a little help from AI.

IMPORTANT: This is a massive work in progress. I am working daily to polish every feature, so expect some things to be unfinished for now while I continue building the engine.
