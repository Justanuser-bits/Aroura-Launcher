Feedback, Support & Bug Tracking

Does this require an official account?

Aurora supports both Microsoft official login for skins and servers, and Offline mode for local play.

Is the built-in X-Ray detectable?

Our X-Ray uses custom resource-pack injection. It is undetectable by standard client-side checks.

Why does it install Java 21?

Modern versions require it. Aurora provisions a portable JDK 21 automatically to keep your system clean.

Fixing Mod Conflicts

Check the Mod Manager. Ensure you aren't mixing Fabric and Forge versions for the same game session.

Optimizing RAM Usage

2GB-4GB is recommended. Set this in the Settings tab to ensure smooth gameplay on older hardware.

Fullbright is flickering?

Custom shaders usually override the lightmap. Disable external shaders to use built-in Fullbright.

Privacy Policy

01

Data Sovereignty

All authentication tokens, game logs, and account details are stored locally in your `minecraft_data` folder. We do not maintain a central server for user data storage.

02

Authentication

Microsoft logins are performed using official secure OAuth standards. Your password is never seen or stored by the launcher; only a local session token is managed to keep you signed in.

03

Analytics & Tracking

Aurora Launcher does not contain telemetry, tracking scripts, or usage analytics. We believe in a focused experience where your activity remains private to your device.
